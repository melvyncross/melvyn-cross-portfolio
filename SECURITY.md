# Security Policy

## Supported Versions

This is a personal portfolio site. Only the latest deployment is actively maintained.

| Version | Supported |
| ------- | --------- |
| Latest (main branch) | ✅ |
| Older deployments | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it privately using GitHub's [private vulnerability reporting](https://github.com/melvyncross/melvyn-cross-portfolio/security/advisories/new).

Please **do not** open a public issue for security vulnerabilities.

**What to include:**
- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any suggested fixes (optional)

I aim to respond within 7 days and will keep you informed of progress toward a fix.
